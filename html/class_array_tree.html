Detailed Description

Classe représentant un arbre binaire statique.

ArrayTree La classe contient quatre tableaux, parent/filsGauche/filsDroit/data. Quand on ajoute une node, elle vérifie qu'il y a suffisamment de place pour ajouter une node, sinon elle créée un nouveau tableau plus grand et remets les anciens noeuds et le nouveau
