Detailed Description

Classe représentant un arbre binaire dynamique.

La classe construit une racine pour un arbre binaire, à laquelle on rajoute des noeuds par insert ou insertBalanced On peut après lui appliquer les actions sur les arbres binaires, dont le SPR en itératif, récursif, list.
